# Exercice Typescript B

Réaliser une page internet via HTML / CSS et Typescript permettant l'affichage, sous la forme que vous voulez, de contacts (**nom, prénom, email et numéro de téléphone**).
Chaque contact devra être unique via un ID personnel et il devra être possible de suivre le nombre de contacts dans l'application via une variable statique liée à tous les contacts. 

Pour ajouter un contact, vous passerez par un formulaire dont vous récupèrerez l'évènement. Les contacts seront stockés dans un **tableau de contact**. Il devra aussi être possible de pouvoir supprimer un contact, ce qui mettra à jour le compteur de contacts et l'affichage. 
 
Bonus : Il sera possible de choisir comment trier les contacts (par age, id, nom ou prénom)
